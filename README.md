# Study APIR

API Java para estudo de Spring Boot

## Instalação

* Limpar e criar a pasta */target*

```
mvn clean package
```

## Navegação

### Executar a API

-  *Executando* **Maven**

```
mvn spring-boot:run
```

### Documentação da API (Swagger)
- http://localhost:8080/swagger-ui.html
